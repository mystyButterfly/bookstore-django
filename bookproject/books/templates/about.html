About Our Book Store

Welcome to our Book Store Web Application! This platform offers a seamless experience for both customers and administrators, built using the Django framework.

Key Features

Home Page

  • Displays a list of books available in the store.
  • Each book is presented with its title, description, image, and price.
  • Users can click on a book to view more details or add it to their cart.

Cart Page

  • Displays a detailed view of the selected book(s) in the shopping cart.
  • Shows the book's title, description, price, and a button to complete the purchase.
  • When the user clicks on the "Buy NOW" button, a polite message is displayed to confirm the purchase (with future enhancements like animations and modal windows).

Admin Panel

  • The administrator of the store has access to an admin interface (provided by Django) where they can manage the store's inventory.
  • The admin can add new books, edit existing book details, and delete books.
  • The admin can input data such as the book's title, description, price, and image URL, allowing the store's catalog to stay up to date.

Responsive Design

  • The website is mobile-friendly, with CSS styling adjustments that ensure proper display on mobile devices (e.g., flexible layouts, stacked elements).
  • The images, price, and buttons adjust dynamically based on the screen size, ensuring a seamless user experience across all devices.

Detailed Description of the Admin Panel Features

The admin panel is one of the core components of this project. Django’s built-in admin interface is used to manage the books and update the store’s inventory. The admin panel enables the following functionalities:

Add New Books

  • The administrator can add new books by filling out a form with the following fields:
    • Title: The name of the book (e.g., "Harry Potter - Rowling").
    • Description: A brief description of the book (e.g., "Harry's adventures").
    • Price: The price of the book in USD.
    • Image URL: The URL of the book's cover image.

Edit Existing Books

  • The administrator can update any of the fields for an existing book, including title, description, price, or image URL.

Delete Books

  • The admin can delete books from the inventory when they are no longer available for sale.

View All Books

  • The admin has access to a list view where all books are displayed in a table format. This provides easy access to each book’s details, allowing for quick edits and deletions.
